# dLNk IDE - AI Bridge

AI Bridge system for dLNk IDE project.

## Features
- Multiple AI provider support (FREE providers only)
- Auto-failover system
- Token management
- gRPC client for Antigravity

## Providers (All FREE)
1. Antigravity (Primary)
2. Groq (Secondary)
3. Ollama (Tertiary)
4. Jetski (Quaternary)

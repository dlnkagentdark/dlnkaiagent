# dLNk IDE

**AI-Powered Integrated Development Environment**

## Overview

dLNk IDE is a next-generation integrated development environment built on VSCode foundation, enhanced with AI capabilities.

## Project Structure

```
dlnkaiagent/
├── admin-console/      # Admin dashboard
├── backend/            # Backend API services
├── deploy/             # Deployment configs
├── docs/               # Documentation
├── extension/          # VSCode extension
├── releases/           # Release builds
├── security/           # Security modules
├── source-files/       # Core source files
├── telegram-bot/       # Telegram bot
├── tests/              # Test suites
├── ui-design/          # UI/UX design
└── vscode-fork/        # VSCode fork
```

## Getting Started

```bash
git clone https://github.com/dlnkagentdark/dlnkaiagent.git
cd dlnkaiagent
cp .env.example .env
npm install
npm run dev
```

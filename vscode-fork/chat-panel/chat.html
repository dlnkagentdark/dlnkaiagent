Can you explain how async/await works in Python?

dLNk AI Just now

Async/Await in Python allows you to write asynchronous code that can handle multiple operations concurrently without blocking.

Here's a simple example:

python 
import asyncio

async def fetch_data(url):
    print(f"Fetching {url}...")
    await asyncio.sleep(2)  # Simulate network delay
    return f"Data from {url}"

async def main():
    # Run multiple tasks concurrently
    tasks = [
        fetch_data("api/users"),
        fetch_data("api/posts"),
        fetch_data("api/comments")
    ]
    results = await asyncio.gather(*tasks)
    for result in results:
        print(result)

# Run the async function
asyncio.run(main())

Key concepts:

  • async def - Defines an asynchronous function
  • await - Pauses execution until the awaited task completes
  • asyncio.gather() - Runs multiple coroutines concurrently

# Assets Directory

This directory contains static assets for the dLNk Admin Console.

## Structure

### icons/
Contains application icons and logos in various formats and sizes:
- `dlnk-logo.ico` - Windows icon format
- `dlnk-logo.svg` - Scalable vector format
- `dlnk-logo-16.png` through `dlnk-logo-512.png` - PNG icons in various sizes

### fonts/
Reserved for custom fonts (currently empty, uses system fonts)

## Usage

The application automatically loads the window icon from `icons/dlnk-logo.ico` on startup. Additional icons can be added here for future features such as toolbar buttons or status indicators.

## Source

Icons are sourced from the dLNk IDE ui-design directory to maintain consistent branding across all dLNk applications.
